## 建立新專案

專案是您數位體驗的基礎。它代表您的場館、活動或訪客將探索的內容集合。

:::info 免費方案限制
免費方案用戶最多可建立 3 個專案。升級至 Premium 可擁有最多 15 個專案。
:::

## 逐步指南

### 步驟 1：前往「我的專案」

登入後，點擊導航選單中的「我的專案」進入專案儀表板。

![前往專案](/Image/docs/navigate-projects.png "ExperienceQR 儀表板")

### 步驟 2：點擊「建立新專案」

點擊專案頁面右上角的「新增專案」按鈕。

![建立專案按鈕](/Image/docs/create-project-button.png "建立專案")

### 步驟 3：填寫專案詳情

填寫專案建立表單的以下資訊：

- **專案名稱** - 為您的場館或體驗取一個清晰、描述性的名稱
- **描述** - 簡短概述，幫助訪客了解他們將發現的內容
- **封面圖片** - 代表您專案的吸睛圖片（建議：1200x630像素）
- **內容模式** - 選擇內容的顯示方式（單一、列表、網格、群組或內嵌）

![專案表單](/Image/docs/project-form.png "專案詳情表單")

### 步驟 4：儲存並繼續

點擊「建立專案」儲存。您將被導向專案編輯器，可以開始新增內容。

:::tip 快速預覽
發布前使用預覽按鈕查看訪客將看到的專案樣貌！
:::

## 後續步驟

建立專案後，您可以：

- 新增包含描述和圖片的內容項目
- 設定 AI 助理設定和知識庫
- 設定多語言翻譯
