## 使用範本

### 步驟 1：瀏覽範本

1. 前往「我的專案」
2. 點擊「新增專案」
3. 選擇「從範本開始」
4. 按類別瀏覽

### 步驟 2：預覽範本

點擊任何範本以預覽：

- 內容結構
- 範例內容項目
- AI 設定
- 顯示模式

### 步驟 3：匯入範本

點擊「使用範本」以根據範本建立新專案。

:::info 範本匯入
匯入範本會建立新專案。您的訂閱限制適用——免費方案最多允許 3 個專案。
:::

### 步驟 4：自訂

匯入後，自訂範本：

- 用您自己的內容替換範例內容
- 更新圖片
- 修改 AI 指令
- 根據需要調整設定

## 範本功能

每個範本包含：

- **優化的內容模式** - 適合該場館類型的最佳版面
- **範例內容結構** - 建議的組織方式
- **AI 設定** - 量身定制的 AI 指令
- **歡迎訊息** - 可自訂的問候語

## 從範本建立 vs 從頭開始

| 方法 | 最適合 |
|------|--------|
| **範本** | 快速開始、標準場館類型、新用戶 |
| **從頭開始** | 特殊需求、自訂版面、有經驗的用戶 |

:::tip 混合使用
您可以從範本開始，然後新增、移除或重新組織內容以符合您的確切需求。
:::
