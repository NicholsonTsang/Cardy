## 批量匯入

使用 Excel 試算表一次匯入多個內容項目，節省時間。

## Excel 範本

下載我們的 Excel 範本開始：

1. 前往「我的專案」
2. 點擊「批量匯入」
3. 下載範本檔案

![批量匯入按鈕](/Image/docs/bulk-import-button.png "批量匯入")

## 範本結構

Excel 範本包含以下欄位：

| 欄位 | 描述 | 必填 |
|------|------|------|
| 名稱 | 項目標題 | 是 |
| 描述 | 項目描述 | 是 |
| 類別 | 群組名稱（群組模式用） | 否 |
| 圖片 URL | 圖片連結 | 否 |
| AI 知識 | 額外 AI 背景資訊 | 否 |
| 順序 | 顯示順序（數字） | 否 |

:::tip 圖片 URL
圖片必須是可公開存取的 URL。請先將圖片上傳到圖床服務，再將 URL 填入試算表。
:::

## 匯入過程

### 步驟 1：準備您的資料

使用您的內容填寫 Excel 範本：

- 每個內容項目一行
- 保持文字簡潔清晰
- 確保圖片 URL 有效

### 步驟 2：上傳檔案

1. 在您的專案上點擊「批量匯入」
2. 選擇您完成的 Excel 檔案
3. 檢閱預覽

![匯入預覽](/Image/docs/bulk-import-preview.png "匯入預覽")

### 步驟 3：確認匯入

檢閱匯入預覽：

- 確認項目數量符合預期
- 驗證範例內容正確
- 解決任何驗證錯誤

### 步驟 4：完成匯入

點擊「匯入」將所有項目新增到您的專案。大量匯入可能需要一些時間。

:::warning 訂閱限制
批量匯入遵守訂閱限制。免費方案用戶最多可擁有 3 個專案。如果匯入會超出限制，您需要升級或減少項目。
:::

## 處理錯誤

常見匯入錯誤及解決方案：

### 缺少必填欄位
- 確保名稱和描述欄位已填寫
- 從試算表中移除空白行

### 無效的圖片 URL
- 驗證 URL 可公開存取
- 檢查 URL 是否有拼寫錯誤
- 確保圖片為支援的格式（JPG、PNG、WebP）

### 重複名稱
- 每個項目名稱在專案中必須唯一
- 為重複名稱添加區分詳情

## 最佳實踐

1. **從小規模開始** - 先用少量項目測試
2. **驗證 URL** - 匯入前檢查所有圖片連結
3. **一致的格式** - 保持描述長度和風格相似
4. **備份原檔** - 保留您的 Excel 檔案副本
5. **匯入後檢閱** - 驗證所有內容正確匯入

## 更新匯入的內容

匯入後，您可以個別編輯項目：

- 點擊任何項目開啟編輯器
- 進行變更並儲存
- 變更立即生效
