## 批量匯入

使用 Excel 試算表一次匯入多個內容項目，節省時間。批量匯入支援完整專案創建，包含所有設定、內容和翻譯。

## Excel 範本

下載我們的 Excel 範本開始：

1. 前往「我的專案」
2. 點擊「批量匯入」
3. 下載範本檔案

![批量匯入按鈕](/Image/docs/bulk-import-button.png "批量匯入")

## 範本結構

Excel 範本包含多個工作表：

### 卡片（專案）工作表

| 欄位 | 說明 | 必填 |
|------|------|------|
| Name | 專案名稱 | 是 |
| Description | 專案概覽 | 是 |
| Content Mode | single/list/grid/grouped/inline | 是 |
| Original Language | 源語言代碼 | 否 |
| AI Instruction | AI 行為指南 | 否 |
| AI Knowledge Base | 通用 AI 背景 | 否 |
| AI Welcome General | 專案層級問候語 | 否 |
| AI Welcome Item | 項目層級問候語範本 | 否 |
| Is Grouped | 啟用分組顯示 | 否 |
| Group Display | collapsed/expanded | 否 |
| Max Scans | 總掃描限制 | 否 |
| Daily Scan Limit | 每日限制 | 否 |

### 內容工作表

| 欄位 | 說明 | 必填 |
|------|------|------|
| Name | 項目標題 | 是 |
| Description | 項目描述 | 是 |
| Parent Name | 子項目的父項目名稱 | 否 |
| Image URL | 圖片連結 | 否 |
| AI Knowledge Base | 項目專屬 AI 背景 | 否 |
| Order | 顯示順序（數字） | 否 |
| Translations JSON | 預翻譯內容 | 否 |

:::tip 圖片網址
圖片必須是可公開存取的網址。先將圖片上傳到主機服務，然後在試算表中包含網址。
:::

## 匯入流程

### 步驟 1：準備資料

在 Excel 範本中填入內容：

- 每個內容項目一列
- 保持文字簡潔清晰
- 確保圖片網址有效
- 為階層內容包含父項目名稱

### 步驟 2：上傳檔案

1. 在「我的專案」頁面點擊「批量匯入」
2. 選擇已完成的 Excel 檔案
3. 檢閱匯入預覽

![匯入預覽](/Image/docs/bulk-import-preview.png "匯入預覽")

### 預覽資訊

預覽顯示：
- 專案設定（模式、語言、AI 狀態）
- 內容項目數量
- 包含的翻譯
- 分組結構狀態

### 步驟 3：確認匯入

檢閱匯入預覽：

- 檢查設定徽章（模式、語言、AI 已啟用）
- 驗證內容項目數量
- 查看包含的翻譯（如有）
- 解決任何驗證錯誤

### 步驟 4：完成匯入

點擊「匯入」創建專案。大型匯入可能需要一些時間。

:::warning 訂閱限制
批量匯入遵守訂閱限制。免費方案用戶總共最多可有 3 個專案。如果超出限制，匯入將失敗。
:::

## 匯出專案

您可以將現有專案匯出為 Excel：

1. 開啟您的專案
2. 點擊「匯出為 Excel」
3. 下載產生的檔案

匯出包含：
- 所有專案設定
- 含階層的內容項目
- AI 設定
- 翻譯（如有）

## 處理錯誤

常見匯入錯誤及解決方案：

### 缺少必填欄位
- 確保名稱和描述欄位已填寫
- 移除試算表中的空白列

### 無效圖片網址
- 驗證網址可公開存取
- 檢查網址拼寫錯誤
- 確保圖片為支援格式（JPG、PNG、WebP）

### 重複名稱
- 專案內每個項目名稱必須唯一
- 為重複名稱新增區分細節

### 無效內容模式
- 使用確切值：single、list、grid、grouped、inline
- 區分大小寫

## 保留翻譯

匯入先前匯出的 Excel 檔案時：

- **Translations JSON** 欄位保留所有翻譯內容
- 匯出中包含的所有語言都會被匯入
- 翻譯狀態會被保留

:::tip 往返編輯
匯出 → 在 Excel 中編輯 → 重新匯入可保留所有翻譯和設定。適合批量文字更新。
:::

## 最佳實踐

1. **從小開始** - 先用幾個項目測試
2. **驗證網址** - 匯入前檢查所有圖片連結
3. **一致格式** - 保持描述長度和風格相似
4. **備份原檔** - 保留 Excel 檔案副本
5. **匯入後檢閱** - 驗證所有內容正確匯入
6. **檢查 AI 欄位** - 確保 AI 知識庫內容準確

## 更新匯入的內容

匯入後，您可以個別編輯項目：

- 點擊任何項目開啟編輯器
- 進行變更並儲存
- 變更立即生效
- 重新匯出以更新您的 Excel 備份
