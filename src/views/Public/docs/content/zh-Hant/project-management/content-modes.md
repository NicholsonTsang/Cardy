## 了解內容模式

內容模式決定您的內容如何向訪客展示。選擇最適合您場館類型和內容結構的模式。

## 可用模式

### 單一模式

適合以單一主要內容為重點的體驗。

- **最適合**：產品展示、特別展覽、活動資訊
- **版面**：全螢幕內容，呈現最大細節

### 列表模式

垂直捲動的內容項目列表。

- **最適合**：菜單、服務列表、簡單目錄
- **版面**：清晰列表，含圖片和描述

### 網格模式

以縮圖網格呈現的視覺畫廊版面。

- **最適合**：藝術畫廊、照片集、產品目錄
- **版面**：響應式網格，含圖片預覽

### 群組模式

內容按可展開的類別組織。

- **最適合**：博物館、多樓層場館、分類菜單
- **版面**：可摺疊區段，含巢狀項目

### 內嵌模式

所有內容可見的連續捲動。

- **最適合**：時間軸體驗、故事驅動內容
- **版面**：無縫垂直流動

## 選擇正確模式

| 場館類型 | 建議模式 |
|----------|---------|
| 博物館 | 群組 |
| 餐廳 | 列表 |
| 畫廊 | 網格 |
| 活動 | 單一 |
| 導覽 | 內嵌 |

:::tip 發布前預覽
使用預覽功能查看每種模式如何顯示您的內容，再做最終決定。
:::

## 變更內容模式

變更專案的內容模式：

1. 開啟專案設定
2. 前往顯示設定
3. 選擇新的內容模式
4. 預覽變更
5. 滿意後儲存

:::warning 內容重新組織
切換到群組模式時，您可能需要將項目重新組織成類別。從群組模式切換出去時，子項目將變成頂層項目。
:::
