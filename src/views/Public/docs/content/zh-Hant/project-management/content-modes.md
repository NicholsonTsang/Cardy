## 了解內容模式

內容模式決定您的內容如何呈現給受眾。選擇最適合您內容類型和結構的模式。

## 可用模式

FunTell 提供 **4 種內容顯示模式**：

### 單一模式

適合聚焦於一個主要內容的體驗。

- **最適合**：產品展示、特別展覽、活動資訊、公告
- **版面**：全螢幕內容，最大細節
- **AI 行為**：深入介紹特色項目

### 列表模式

垂直捲動的內容項目列表。

- **最適合**：菜單、服務列表、資源目錄、連結彙整
- **版面**：乾淨的垂直列表配圖片和描述
- **AI 行為**：幫助瀏覽選項，回答任何項目的問題

### 網格模式

雙欄縮圖網格的視覺畫廊版面。

- **最適合**：藝術畫廊、照片集、產品目錄
- **版面**：響應式雙欄網格配圖片預覽
- **AI 行為**：探索畫廊，根據興趣建議項目

### 卡片模式

全寬卡片版面，適合精選內容。

- **最適合**：精選項目、新聞動態、重點內容、作品集
- **版面**：大型卡片配醒目圖片
- **AI 行為**：引導瀏覽精選內容，提供推薦

## 選擇正確的模式

| 使用場景 | 推薦模式 | 原因 |
|----------|---------|------|
| 產品目錄 | 列表或網格 | 方便瀏覽品項 |
| 博物館 / 畫廊 | 網格 | 視覺瀏覽體驗 |
| 餐廳菜單 | 列表 | 清晰的菜單瀏覽 |
| 教育課程 | 列表 | 結構化課程順序 |
| 活動 / 會議 | 單一或列表 | 聚焦資訊或場次列表 |
| 知識庫 | 列表 | 有組織的參考內容 |
| 作品集 | 卡片 | 展示精選作品 |
| 新聞 / 部落格 | 卡片 | 突出文章或章節 |

:::tip 發布前預覽
使用預覽功能查看每種模式如何顯示您的內容，再做最終決定。
:::

## 內容分組

除了顯示模式，您還可以啟用**內容分組**來將項目組織成類別。

### 啟用分組

1. 前往專案設定
2. 開啟**「啟用分組」**
3. 創建父項目（類別）
4. 將內容項目放入父項目下

### 分組與模式組合

| 模式 | 搭配分組 | 範例 |
|------|---------|------|
| 列表 | 類別下顯示巢狀項目 | 菜單分區（前菜、主菜、甜點） |
| 網格 | 類別標題上方顯示網格 | 畫廊樓層（一樓、二樓、三樓） |
| 卡片 | 分組精選區塊 | 新聞類別（科技、體育、文化） |

### 分組顯示選項

| 選項 | 行為 |
|------|------|
| **展開** | 類別標題下顯示所有項目 |
| **摺疊** | 點擊類別才顯示項目 |

## 不同模式中的 AI 助理

AI 助理會根據當前內容模式調整其行為：

### 導覽頁面

- 出現**通用助理**（浮動按鈕）
- 幫助瀏覽、推薦、一般問題
- 在所有列表/網格/卡片視圖中可用

### 詳情頁面

- 出現**項目助理**（內嵌按鈕）
- 提供關於特定項目的深入資訊
- 使用項目專屬的知識庫和歡迎訊息

| 模式 | AI 行為 |
|------|---------|
| 單一 | 深入介紹特色項目 |
| 列表 | 幫助瀏覽和篩選選項 |
| 網格 | 探索畫廊，建議項目 |
| 卡片 | 引導瀏覽精選內容 |

## 更改內容模式

更改專案的內容模式：

1. 開啟您的專案
2. 前往**設定**分頁
3. 找到**顯示設定**
4. 選擇新的內容模式
5. 預覽變更
6. 滿意後儲存

:::warning 內容重組
啟用分組時，您可能需要將項目重新組織成類別。停用分組時，父項目將與其子項目一起顯示。
:::

## 最佳實踐

### 模式選擇

- **考慮您的內容類型** - 視覺內容 → 網格，文字為主 → 列表
- **思考訪客旅程** - 他們會如何瀏覽？
- **用真實內容測試** - 確定前先預覽

### 內容組織

- **使用一致的命名** - 清晰的標題幫助訪客導覽
- **邏輯排序** - 最重要的項目優先或按邏輯順序
- **適當的圖片** - 圖片風格與所選模式相符

### 手機體驗

- **所有模式都針對手機優化**
- **在手機上測試** - 您的訪客主要使用手機
- **考慮觸控目標** - 確保項目容易點擊
